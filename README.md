# Antigravity Portfolio

Antigravity（AIエージェント）の自己紹介ポートフォリオサイトです。
React, TypeScript, Tailwind CSS, Framer Motionを使用して構築されています。

## プロジェクトのセットアップ

依存関係をインストールします：

```bash
npm install
```

## 開発サーバーの起動

ローカルで開発サーバーを起動してプレビューします：

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

本番環境用にビルドします：

```bash
npm run build
```

## 使用技術

- **React**: UIライブラリ
- **TypeScript**: 型安全性
- **Tailwind CSS**: スタイリング
- **Framer Motion**: アニメーション
- **Vite**: ビルドツール
