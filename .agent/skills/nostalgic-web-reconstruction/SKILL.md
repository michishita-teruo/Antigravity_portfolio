# Nostalgic Web Reconstruction Skill

このスキルは、現代のクリーンなWebデザインの対極にある、90年代特有の「手作り感」「粗削りさ」「特定の技術的制約」を意図的に再現するためのガイドラインとツールを提供します。

## 🎨 コア・スタイル

### 1. タイポグラフィ
- **MS Pゴシックスタック**: 日本のレトロWebの象徴。
  ```css
  font-family: "MS PGothic", "MS Pゴシック", "MS Gothic", "Osaka", "ヒラギノ角ゴ Pro W3", sans-serif;
  ```
- **アンチエイリアスの抑制**: ブラウザの標準レンダリングに任せ、あえてカクカクした質感を出す。

### 2. カラーパレット
- **原色の多用**: #0000FF (青), #FF0000 (赤), #FFFF00 (黄)。
- **Windows Classic**: #C0C0C0 (銀色) の背景と、凸凹感を出す枠線（border-outset/inset）。

### 3. アビリティ（特殊演出）
- **タイガーストライプ**: `repeating-linear-gradient` を使用した工事中模様。
- **ネオン点滅**: `framer-motion` を使用した blink エフェクトの再現。
- **マーキー**: CSS Animation による無限スクロールテキスト。

## 🛠 実装の手順

1. **基本構造の破壊**: グリッドやフレックスボックスをあえて不自然に使い、当時の「テーブルレイアウト」に近い重なりを演出する。
2. **アクセスカウンターの設置**: Firestoreと連携しつつ、見た目は7セグメントLED風の画像またはフォントを使用する。
3. **リンク導線の意図的な不便化**: 目立たない隠しリンクや、キリ番報告などのコミュニティ要素の配置。

## 🚫 禁止事項
- **洗練されたマージン**: 当時はピクセル単位で詰め込まれていた。
- **滑らかなグラデーション**: 256色時代のディザリング感を意識し、フラットか極端なグラデーションのみ許容。
- **Google Fonts (Modern)**: レトロセクション内では、システムフォント以外を使用してはならない。
